Project 7--DOM and JS

JavaScript Questions

1. Why is textContent safer than innerHTML?

Answer: textContent is safer because it treats content as plain text instead of HTML, so it does not insert or run HTML or scripts.

2. How are HTML elements connected to JavaScript variables using DOM selection?

Answer: HTML elements are connected to JavaScript variables by selecting them with DOM methods like getElementById() or querySelector() and storing them in variables.

3. What would happen to the text box with id="text" if I added this line to the code above it: text.textContent = "Hello!"

Answer: The text inside the element with the id of "text" would change to Hello!
